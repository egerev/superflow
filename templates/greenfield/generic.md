# Generic Greenfield Template

For stacks without a specific template, create a minimal structure.

## Directory Structure
```
├── src/
│   └── .gitkeep
├── tests/
│   └── .gitkeep
├── docs/
│   └── .gitkeep
├── .gitignore
├── README.md
└── CLAUDE.md
```
